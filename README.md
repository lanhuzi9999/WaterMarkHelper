# WaterMarkHelper<br>
水印助手，比较实用的工具类，只需要在application中初始化便可以在app所有界面添加水印：<br>
mWaterMarkHelper = new WaterMarkHelper();<br>
mWaterMarkHelper.installWaterMark(this, 0, "Debug测试版", 25 * 3f, Color.BLUE & 0x19ffffff);<br>
